# PRD: 루키스의 그림판 (lukis-drawing-board)

## 목표

강의용 드래그 앤 드롭 다이어그램 도구 — 좌측 팔레트에서 요소를 꺼내 React Flow 칠판에 배치하고 JSON으로 저장·공유한다.

## 사용자

- 주 사용자: AI 강의 진행자
- 보조: 학습 자료를 정리하려는 학습자

## 핵심 기능

1. 좌측 팔레트에서 요소(아이템/컨테이너/연결선) 드래그하여 칠판에 배치
2. 컨테이너로 노드 그룹화(중첩 가능), 4종 연결선으로 노드 연결, 인라인 라벨 수정
3. 우측 프리셋 패널에 칠판 저장/로드, JSON import/export, 사용자 팔레트 추가

## MVP 제외 사항

- 프리셋 썸네일/검색/정렬
- 자동 레이아웃
- 노드 색상 커스터마이즈
- 모바일/터치 지원
- PNG export
- 다중 사용자
- 미니맵
- 키보드 드래그 대안

## 디자인

- 다크 무드, 무광. UI-GUIDE의 anti-slop 가이드 준수.
- 무채색 기반 (`#0a0a0a` page, `#141414` card). 셀렉션 색상만 `ring-violet-400/60` 절제 사용. 참조 이미지의 네온/보라 톤 불채택.
